# IPEDS – Big Data Analytics using Spark & Hive

## Objective
Analyze multi-year higher education enrollment data (2020–2023)
to identify trends, sector-wise differences, and key factors
influencing online program growth.

## Tools
Apache Spark, Hive, Hadoop, SQL, Python, Excel

## Approach
- Ingested and standardized IPEDS datasets across multiple years
- Created Hive tables for structured querying
- Performed Spark-based transformations and aggregations
- Conducted trend and correlation analysis
- Answered business questions using large-scale data processing

## Key Insights
- Online enrollment increased steadily post-2020
- Private institutions showed higher growth rates than public
- Strong correlation observed between program type and enrollment volume
